# 数据介绍
数据都在all文件下，分别是数据结构，计算机组成原理，操作系统和计算机网络的闪卡数据原始结构
文件名字：
1. ds_all.json是数据结构的卡片数据
2. co_all.json是计算机组成原理的卡片数据
3. os_all.json是操作系统的卡片数据
4. cn_all.json是计算机网络的卡片数据

需求：
你可以通过写python来完成这个任务，并且python就生成在json目录即可
1. 先读取原始数据库已经存在的数据，不要破坏之前模拟的数据
2. 读取ds_all.json的数据，然后根据读取数据库原有的信息进行新增
3. 读取co_all.json的数据，然后根据读取数据库原有的信息进行新增
4. 读取os_all.json的数据，然后根据读取数据库原有的信息进行新增
5. 读取cn_all.json的数据，然后根据读取数据库原有的信息进行新增
