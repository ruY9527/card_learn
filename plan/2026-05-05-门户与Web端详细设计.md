### 4.2 注册页 (`/register`) - 新增

**表单字段：**

| 字段 | 必填 | 验证规则 |
|------|------|----------|
| 用户名 | 是 | 3-20字符，字母数字下划线 |
| 密码 | 是 | 6-20字符 |
| 确认密码 | 是 | 与密码一致 |
| 昵称 | 否 | 默认使用用户名 |
| 验证码 | 是 | 4位字符 |

**后端新增接口：**

```
POST /auth/register
Request: { username, password, nickname?, captcha, captchaKey }
Response: { token, user: { userId, username, nickname, avatar } }
```

5. 进度显示（当前/总数）
6. 快捷键支持（空格翻转，1-4选择状态）

## 七、路由守卫逻辑

```typescript
// 路由守卫伪代码
router.beforeEach((to, from, next) => {
  const token = localStorage.getItem('token')
  const userInfo = userStore.userInfo

  // 公开路由
  if (to.path === '/' || to.path === '/login' || to.path === '/register') {
    if (token && userInfo) {
      // 已登录用户访问公开页，重定向到对应首页
      if (isAdmin(userInfo)) {
        next('/dashboard')
      } else {
        next('/learn')
      }
    } else {
      next()
    }
    return
  }

  // 需要登录的路由
  if (!token) {
    next('/login')
    return
  }

  // 已登录，访问登录页则重定向
  if (to.path === '/login' || to.path === '/register') {
    if (isAdmin(userInfo)) {
      next('/dashboard')
    } else {
      next('/learn')
    }
    return
  }

  // 根据角色决定可访问的路由
  if (isAdmin(userInfo)) {
    // admin 可访问所有路由
    next()
  } else {
    // 普通用户只能访问 /learn/* 路由
    if (to.path.startsWith('/learn')) {
      next()
    } else {
      // 非 admin 访问管理路由，重定向到学习中心
      next('/learn')
    }
  }
})
```
