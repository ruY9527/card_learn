**创建日期：** 2026-05-06

## 一、系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        智能复习系统                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────┐    ┌─────────────┐    ┌─────────────┐       │
│   │  数据采集层  │ →  │  算法引擎   │ →  │  提醒推送层  │       │
│   │  (历史数据)  │    │ (遗忘+周期) │    │ (Web+iOS)  │       │
│   └─────────────┘    └─────────────┘    └─────────────┘       │
│          ↑                  ↑                                    │
│          └──────────────────┴────────────────────────────────── │
│                        复习计划生成                               │
└─────────────────────────────────────────────────────────────────┘
```

### 1.1 核心因素模型

```
复习优先级得分 = α × 遗忘指数 + β × 艾宾浩斯周期 + γ × 学习习惯 + δ × 难度系数
```

| 因素 | 权重说明 | 数据来源 |
|------|----------|----------|
| **遗忘指数** | 反映卡片被遗忘的风险 | 每次复习时用户选择的状态（重来=高风险） |
| **艾宾浩斯周期** | 参考艾宾浩斯遗忘曲线 | 距离上次复习的天数 vs 理论遗忘点 |
| **学习习惯** | 用户活跃时段和学习节奏 | 历史学习时间分布统计 |
| **难度系数** | 卡片本身的难度等级 | 创建卡片时设置的 difficultyLevel |

## 二、核心功能模块

| 模块 | 功能 | 优先级 |
|------|------|--------|
| **遗忘指数计算** | 基于答题状态、间隔天数、失败次数计算遗忘风险 | P0 |
| **复习优先级排序** | 按遗忘指数+艾宾浩斯周期+学习习惯综合排序 | P0 |
| **学习习惯画像** | 采集活跃时段、每日目标、学习偏好 | P1 |
| **智能提醒推送** | Web 站内通知 + iOS 推送 | P1 |
| **复习计划分散** | 当日量过大时自动分散到未来多天 | P2 |
| **管理端数据面板** | 管理员查看用户复习数据分析 | P2 |

## 四、算法设计

### 4.1 遗忘指数计算

```java
/**
 * 计算卡片遗忘指数 (0-100, 越高越需要复习)
 */
public double calculateForgetIndex(BizUserProgress progress) {
    // 1. 状态权重 (上次答题选择)
    // 重来=80, 模糊=30, 掌握=0
    double statusWeight = 0;
    if (progress.getStatus() == 0) {
        statusWeight = 80;
    } else if (progress.getStatus() == 1) {
        statusWeight = 30;
    }

    // 2. 时间衰减 (基于艾宾浩斯曲线)
    // 假设理论遗忘峰值在 6-7 天
    int daysSinceReview = calculateDaysSinceLastReview(progress.getUpdateTime());
    double timeDecay = Math.min(50, daysSinceReview * 7);

    // 3. 失败惩罚 (连续重来的次数)
    double failPenalty = progress.getFailStreak() * 10;

    // 4. 难度加成
    double difficultyBonus = progress.getCard().getDifficultyLevel() * 5;

    return Math.min(100, statusWeight + timeDecay + failPenalty + difficultyBonus);
}
```

### 4.2 艾宾浩斯记忆周期

| 记忆阶段 | 推荐复习时点 |
|----------|--------------|
| 短期记忆 | 20分钟 → 1小时 → 6小时 |
| 长期记忆 | 1天 → 2天 → 4天 → 7天 → 15天 → 30天 |

### 4.3 复习优先级计算

```java
public double calculatePriorityScore(BizUserProgress progress,
                                     BizUserLearningProfile profile,
                                     BizCard card) {
    // 遗忘指数 (权重 40%)
    double forgetIndex = calculateForgetIndex(progress);
    double forgetScore = forgetIndex * 0.4;

    // 艾宾浩斯周期匹配度 (权重 30%)
    double ebbinghausScore = calculateEbbinghausScore(progress) * 0.3;

    // 学习习惯匹配度 (权重 20%)
    double habitScore = calculateHabitScore(progress, profile) * 0.2;

    // 难度系数 (权重 10%)
    double difficultyScore = card.getDifficultyLevel() * 20 * 0.1;

    return forgetScore + ebbinghausScore + habitScore + difficultyScore;
}
```

### 4.4 每日容量分配策略

```java
public List<BizSmartReviewTask> generateDailyReviewPlan(Long userId, int maxDailyCards) {
    List<BizUserProgress> needReview = getProgressNeedingReview(userId);
    List<BizUserProgress> sorted = needReview.stream()
        .sorted(Comparator.comparing(this::calculatePriorityScore).reversed())
        .collect(Collectors.toList());

    List<BizSmartReviewTask> tasks = new ArrayList<>();
    List<BizSmartReviewTask> postponed = new ArrayList<>();

    if (sorted.size() <= maxDailyCards) {
        // 数量足够，全部安排
        for (int i = 0; i < sorted.size(); i++) {
            BizUserProgress p = sorted.get(i);
            tasks.add(createTask(p, i, userId));
        }
    } else {
        // 数量过多，分散安排
        for (int i = 0; i < sorted.size(); i++) {
            BizUserProgress p = sorted.get(i);
            if (i < maxDailyCards) {
                tasks.add(createTask(p, i, userId));
            } else {
                // 计算分散到的日期
                int dayOffset = (i - maxDailyCards) / maxDailyCards + 1;
                postponed.add(createPostponedTask(p, dayOffset, userId));
            }
        }
    }

    // 保存任务到数据库
    saveTasks(tasks);
    saveTasks(postponed);

    return tasks;
}
```
